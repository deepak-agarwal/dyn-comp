Border
xs: Represents an extra-small border radius.
sm: Represents a small border radius.
md: Represents a medium border radius.
lg: Represents a large border radius.
xl: Represents an extra-large border radius.
2xl: Represents a twice-extra-large border radius.
3xl: Represents a thrice-extra-large border radius.
full: Represents a full border radius, creating a circular or fully rounded appearance.
