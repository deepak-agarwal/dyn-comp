Typography
To manage Typography options, the tokens object supports the following keys:
fonts (font families)
fontSizes
fontWeights
lineHeights
letterSpacings

```json
{
  "properties": [
    {
      "abbreviation": "FontFamily",
      "cssEquivalent": "font-family",
      "themeKey": "fonts"
    },
    {
      "abbreviation": "FontSize",
      "cssEquivalent": "font-size",
      "themeKey": "fontSizes"
    },
    {
      "abbreviation": "fontWeight",
      "cssEquivalent": "font-weight",
      "themeKey": "fontWeights"
    },
    {
      "abbreviation": "lineHeight",
      "cssEquivalent": "line-height",
      "themeKey": "lineHeights"
    },
    {
      "abbreviation": "letterSpacing",
      "cssEquivalent": "letter-spacing",
      "themeKey": "letterSpacings"
    },
    {
      "abbreviation": "textAlign",
      "cssEquivalent": "text-align",
      "themeKey": "-"
    },
    {
      "abbreviation": "fontStyle",
      "cssEquivalent": "font-style",
      "themeKey": "-"
    },
    {
      "abbreviation": "textTransform",
      "cssEquivalent": "text-transform",
      "themeKey": "-"
    },
    {
      "abbreviation": "textDecoration",
      "cssEquivalent": "text-decoration",
      "themeKey": "-"
    }
  ]
}
```
