## Space

The space key allows you to customize the global spacing and sizing scale for your project. By default, these spacing values can be referenced by the padding, margin, and top, left, right, and bottom props.

```json
{
  "properties": [
    {
      "abbreviation": "m",
      "cssEquivalent": "margin",
      "themeKey": "space"
    },
    {
      "abbreviation": "mt",
      "cssEquivalent": "margin-top",
      "themeKey": "space"
    },
    {
      "abbreviation": "mr",
      "cssEquivalent": "margin-right",
      "themeKey": "space"
    },
    {
      "abbreviation": "mb",
      "cssEquivalent": "margin-bottom",
      "themeKey": "space"
    },
    {
      "abbreviation": "ml",
      "cssEquivalent": "margin-left",
      "themeKey": "space"
    },
    {
      "abbreviation": "mx",
      "cssEquivalent": "margin-left and margin-right",
      "themeKey": "space"
    },
    {
      "abbreviation": "my",
      "cssEquivalent": "margin-top and margin-bottom",
      "themeKey": "space"
    },
    {
      "abbreviation": "p",
      "cssEquivalent": "padding",
      "themeKey": "space"
    },
    {
      "abbreviation": "pt",
      "cssEquivalent": "padding-top",
      "themeKey": "space"
    },
    {
      "abbreviation": "pr",
      "cssEquivalent": "padding-right",
      "themeKey": "space"
    },
    {
      "abbreviation": "pb",
      "cssEquivalent": "padding-bottom",
      "themeKey": "space"
    },
    {
      "abbreviation": "pl",
      "cssEquivalent": "padding-left",
      "themeKey": "space"
    },
    {
      "abbreviation": "px",
      "cssEquivalent": "padding-left and padding-right",
      "themeKey": "space"
    },
    {
      "abbreviation": "py",
      "cssEquivalent": "padding-top and padding-bottom",
      "themeKey": "space"
    }
  ]
}
```
